\# SAAS SETUP WIZARD



This document defines the onboarding process for new companies

using the SabraFlow SaaS platform.



Every new company must go through a setup wizard.



The wizard allows the company to configure its business structure.



---



\# STEP 1: COMPANY CREATION



The user creates a new company.



Required fields:



Company Name  

Country  

Currency  

Language  



Example:



Company Name: Atlas Sabra  

Country: Morocco  

Currency: MAD  

Language: Arabic  



---



\# STEP 2: BRANCH CREATION



The company defines its branches.



Each branch includes:



Branch Name  

Branch Type  



Examples:



Wholesale  

Retail  

Supplies  



Companies may create multiple branches.



---



\# STEP 3: INVENTORY STRUCTURE



The company chooses its inventory model.



Options:



Separate inventory per branch



or



Shared inventory across branches



---



\# STEP 4: CASH STRUCTURE



The company chooses its cash register structure.



Options:



Separate cash register per branch



or



Single company cash register



---



\# STEP 5: INTER-BRANCH OPERATIONS



The company defines how branches interact.



Options:



Branch sales model



Example:

Wholesale sells to Retail.



or



Inventory transfer model



Example:

Inventory moves between branches.



---



\# STEP 6: ADMIN ACCOUNT



The system creates the main administrator account.



Required fields:



Admin Name  

Email  

Password  



The admin will manage users and settings.



---



\# DEFAULT TEMPLATE



The system may provide templates.



Example:



Sabra Business Template



Includes:



Wholesale branch  

Retail branch  

Production system  

Color inventory  



This template matches the workflow used in Moroccan sabra trading.
